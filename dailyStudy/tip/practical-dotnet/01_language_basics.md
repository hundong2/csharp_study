# 01. C# 기본 문법

실무 코드는 결국 변수, 조건문, 반복문, 메서드, 클래스가 조합된 형태입니다. 고급 문법을 보기 전에 이 기본 흐름을 다시 잡아야 합니다.

## 핵심 개념

- `string`, `int`, `double`, `bool`은 자주 쓰는 기본 타입입니다.
- `var`는 오른쪽 값을 보고 타입을 추론합니다.
- `if`는 조건 분기입니다.
- `foreach`는 컬렉션을 순회합니다.
- 메서드는 반복되는 로직에 이름을 붙이는 방법입니다.
- 클래스는 상태와 동작을 묶습니다.

## 실무에서 왜 필요한가

기초 문법이 흔들리면 JSON DTO, API 서비스, Repository, Background Job 같은 구조를 읽을 때 매번 멈추게 됩니다. 복잡한 코드를 읽을 때도 먼저 변수, 분기, 반복, 메서드 호출 흐름만 따라가면 이해가 쉬워집니다.
