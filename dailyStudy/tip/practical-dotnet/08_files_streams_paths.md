# 08. File, Stream, Path

파일 처리와 경로 조합은 배치, 로그, 임시 파일, 업로드 처리에서 자주 사용합니다.

## 핵심 개념

- `Path.Combine`: OS에 맞는 경로를 조합합니다.
- `File.WriteAllText`: 작은 텍스트 파일을 씁니다.
- `FileStream`: 큰 파일을 스트림으로 처리합니다.
- `using`: 파일 핸들을 안전하게 닫습니다.

## 실무에서 왜 필요한가

파일 핸들을 닫지 않으면 Windows에서 파일이 잠기거나, 서버에서 리소스 누수가 발생할 수 있습니다.
